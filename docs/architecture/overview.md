# 项目概览

## 定位

- 本仓库是 AI-first 全栈项目模板
- 默认面向单开发者配合 `GPT-5.4 / Codex` 持续开发
- 默认采用 `LangGraph-ready` 架构，但具体产品可按需精简或替换 agent 层
- 整体原则为重后端、轻前端

## 交付节奏

- `MVP` 优先支持 `Web`
- 后续可扩展到 `iOS`、小程序或桌面端
- 复用重点应放在领域模型、接口契约、agent 能力与设计 tokens，而不是 UI 代码完全共用

## 当前工程方向

- 工程组织采用 `apps/` 与 `packages/` 分层
- `PostgreSQL` 为固定数据库
- agent 工作流、共享领域能力和设计系统应优先作为长期资产沉淀
- 多人团队流程默认不内建，按单开发者高效率协作优化
